## 🚀 Inicio Rápido

### Desarrollo Local
1.  Copiar `.env.example` a `.env` en `backend/`.
2.  Instalar dependencias: `npm install`.
3.  Iniciar: `npm run dev`.

### Actualización
Ejecutar `ACTUALIZAR.bat` en Windows.
